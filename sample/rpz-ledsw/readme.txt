RPZ-IR-Sensor基板用サンプルプログラム

【概要】
LED点灯、スイッチ入力をするプログラムです。
C++で記述したRaspbian Jessieでコンパイルしたバイナリとソースファイル一式、
および、Python3で記述したスクリプトです。


【ファイル構成 Python3】
rpz_ledsw.py : Python3スクリプト

【使い方 Python3】
ターミナルを開いて以下のように実行してください。
Python3 rpz_ledsw.py

SW1(赤)もしくはSW2(黒)を押すとLED1, 2, 3, 4が点灯します。
プログラムを終了するにはキーボードからCtrl+Cを入力してください。


【ファイル構成 C++】
rpz-ledsw : バイナリ実行ファイル
main.cpp : メインルーチン
Makefile : g++用メイクファイル

【使い方 C++】
ターミナルを開いて以下のように実行してください。
./rpz-ledsw

SW1(赤)もしくはSW2(黒)を押すとLED1, 2, 3, 4が点灯します。
プログラムを終了するにはキーボードからCtrl+Cを入力してください。
ご自身でmain.cppを変更した際は、makeコマンドを実行するとコンパイルされ、
rpz-ledswが更新されます。
